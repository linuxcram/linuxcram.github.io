How to get scripts on your Linux box?
To download "script.sh" change the output file name in the commands accordingly to the real script name.

Remember always to set executable permissions if missing, eg. "chmod +x script.sh".

From web browser:
open the link and copy the code into VIM.

Explanation: after that the text has been copied from the web page, go on the Linux SSH-terminal and...
# type vim script.sh # press i # press SHIFT+INS # press ESC # type :wq # press ENTER


From Linux Command Line:
Use one of the commands, replacing the word "LINK" with the real link of the script, change the output file name accordingly to the real script name:

curl -Lks LINK -o script.sh

OR

wget LINK -O script.sh
