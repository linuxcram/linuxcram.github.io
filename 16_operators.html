 Bash scripting operators
Operators are used in Bash scripts to determine whether a condition is true or false.
Upon on their result different action can be executed.

Arithmetic Comparison Operators
-lt      <
-gt      >
-le      <=
-ge      >=
-eq      ==
-ne      !=

eg. script compare.sh
#!/bin/bash
# define values
num1=10
num2=20
# conditional statement
if [ $num1 -eq $num2 ]; then
echo "The two values are equal"
else
echo "The two values are NOT equal"
fi

# make executable chmod u+x compare.sh
# execute ./compare.sh

String Comparison Operators
=      equal
!=      not equal
<      less than
>      greater than
-n s1      string s1 is not empty
-z s1      string s1 is empty

eg. script comparestrings.sh
#!/bin/bash
# define first string
string1="Hello"
# define second string
string2="World"
# perform conditional check and print corresponding result
if [ $string1 = $string2 ]; then
echo "The two strings are equal"
else
echo "The two strings are NOT equal"
fi

# make executable chmod u+x comparestrings.sh # execute ./comparestrings.sh

# output
The two strings are NOT equal

File Testing Operators
-e filename      Check for file existence
-f filename      Check for regular file existence not a directory
-r filename      Check if file is a readable
-s filename      Check if file is nonzero size
-w filename      Check if file is writable
-x filename      Check if file is executable

eg. script checkfile.sh
#!/bin/bash
# define file using ./ to indicate that the file is located in the current directory where the script is run
file="./text1.txt"
# if condition that check if the file exist and it's not a directory, then display the corresponding result
if [ -e $file ]; then
echo "The file $file exists"
else
echo "The file $file does not exists"
fi

# make executable
chmod u+x checkfile.sh
# execute
./checkfile.sh

# output
The file ./text1.txt exists

The echo command accepts text strings between double quotes as argument, and can also read variables values.
Variables are recognized by the $ char at the beginning of their name.

Boolean Operators
Boolean operators are used to test if two or more conditions are true or false.

Operators / function
&& and
|| or
! not equal to

eg.
if [ 3 -ge 1 ] && [ "a" = "a" ]; then echo "yes"; else echo "no"; fi
if [ 3 -eq 1 ] || [ "a" = "a" ]; then echo "yes"; else echo "no"; fi
if [ 3 -le 1 ] || [ "a" = "b" ]; then echo "yes"; else echo "no"; fi

if [ 1 -eq 1 ] && [ "a" = "a" ] || [ "b" = "c" ]; then echo "yes"; else echo "no"; fi
if [ 1 -eq 1 ] && [ "a" = "a" ] && [ "b" = "c" ]; then echo "yes"; else echo "no"; fi
