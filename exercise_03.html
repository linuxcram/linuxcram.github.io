Exercise_03

Create a script named ports.sh that performs the following tasks:

- download, naming the output file name portslist.txt, the content of the web page at the URL: https://www.meridianoutpost.com/resources/articles/well-known-tcpip-ports.php

- generate a new file names portstable.txt, that will contain only the code of the html table showing the ports and their information, from 0 to 995 
Hint: this file must contain from the first line 0
to the first encountered line containing  that is placed after the line 995

- generate a file named ports.csv that contains only the lines containing ports and related information, corresponding to the following fields of the file portstable.txt separated only by #:
Port number#Protocol type(s)#Description#Official/Unofficial/-#

eg.
53#TCP, UDP#DNS (Domain Name System)#Official#
80#TCP#HTTP (HyperText Transfer Protocol) - used for transferring web pages#Official#
411#TCP#Direct Connect Hub port#Unofficial#
513#UDP#Who#-#

BONUS: When done, transfer a copy of the file ports.csv to the local pc and open the file with a worksheet application, choosing the char # as field separator.
