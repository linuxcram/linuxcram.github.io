Special mention : TLCL - The Linux Command Line

Credits to the Author: William Shotts
https://www.linuxcommand.org/tlcl.php
The Linux Command Line is aimed to lead the reader from first terminal keystrokes to writing full programs in Bash, the popular Linux shell (CLI)
"For those looking to master the Linux command line and get an essential understand of the core Linux command line tools, this book is a highly effective and useful guide." Ben Rothke, RSA Conference
https://sourceforge.net/projects/linuxcommand/files/TLCL/19.01/

... and AWTLCL - Adventures with the Linux Command Line
Credits to the Author: William Shotts
A sequel/supplement to the first book, The Linux Command Line (TLCL) with frequent references to it.
https://sourceforge.net/projects/linuxcommand/files/AWTLCL/21.10/
