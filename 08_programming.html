BASH scripting ...and programming?

...what is a programming language?

Which kind of programs usually feature commands, syntax, variables, arrays, loops, iterations, sockets, regular expression matching, I/O management?

Programming languages do that.

Bash does that too.

Bash is a shell and command language, it's the default shell for most Linux distributions.
Created for the GNU Project as a free software replacement for the Bourne shell, first released in 1989, Bash was one of the first programs Linus Torvalds ported to Linux, alongside GCC.
With programming languages, a user can type command programs in an interactive session of an interpreter.
Programs can also be executed in a batch process without human interaction, where the commands execution is chained together.

A scripting language can run commands through an interpreter without compiling.

A program consist of one or more instructions (statements) that perform a specific task.
The Linux command line aka Bash, is used to run commands with arguments, that function as command line parameters to specify the input data for the command.
Within the command script, the passed parameters are accessible using positional parameters.
These range from $0 to $9, where $0 refers to the name of the command itself, $1 to $9 are the first through to the ninth parameter, depending on how many parameters are passed.


Example "have a nice day"


[student@linux ~]$ set have a nice day; echo "$0"; echo -n "$1 $2 $3 $4"; echo ""
bash
have a nice day

# Explanation
$0 = bash since we are writing in the shell, there is not a script name otherwise its name will be assigned
$1 = have
$2 = a
$3 = nice
$4 = day
