[exercise_04]

Create a Bash script named checkdates.sh, write it using functions.

The script must create 4 files named log_DATE.log, but instead of the string DATE every filename will contain an arbitrary date in format YYYYmmdd.

These arbitrary dates must be randomly generated using bash commands that provide random values generation, the dates must be in the range 20000101-20221231.

Every file must contain a different string of 12 characters, including in order 4 capital letters (A-Z), 4 small letters (a-z) and 4 numbers (0-9).

The strings must be created using any bash command that provide random strings generation functions.

Every time the script is executed, it must remove any previous existing file named log_DATE.log, previously created in the current path.

Alway be very careful with disruptive commands such as rm, mv.





Link to sample script:

checkdates.sh


Inspect the result

Check the files generated (read them using cat, more, less)

REMEMBER: DO NOT BLINDLY RUN SHELL COMMANDS OR SCRIPTS FOUND ON THE INTERNET.

Scripts must be read carefully! Check that every command is present on the system.
Commands not found cause errors, often with a suggestion to remediate because Ubuntu is nice:
cal
Command 'cal' not found, but can be installed with:
sudo apt install ncal

The command that install the missing package:
sudo apt install ncal
